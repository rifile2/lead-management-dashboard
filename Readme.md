# Lead Management Dashboard

A mini CRM-style Lead Management Dashboard built as part of a fresher hiring take-home assignment.  
The project demonstrates a full-stack application with frontend, backend, database integration, and deployment.

---

##  Live Project

**Live URL (Frontend + Backend):**  
https://lead-backend-3hku.onrender.com

> The frontend (React) and backend (Node/Express) are deployed together as a **single Render Web Service**.
